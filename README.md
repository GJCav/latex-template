# LateX 模板

## 总体介绍

这个仓库用于分享一些我制作的Latex模板，每一个模板都尽可能做到self-explained、免安装。

这个仓库会随缘更新。

对于T大的朋友们，推荐使用[THU overleaf](https://overleaf.tsinghua.edu.cn)。如果实在是喜欢自己搭建latex环境，推荐在wsl2中安装texlive，使用vscode连接到wsl进行开发，这样编译速度会快一些。
